Z-score Альтмана

Оцените вероятность банкротства компании по 5-факторной модели Альтмана с визуализацией вклада каждого компонента и индикатором зоны риска.

// Altman Z-Score (5-factor model)
Z = 1.2×X1 + 1.4×X2 + 3.3×X3 + 0.6×X4 + 1.0×X5

// где:
X1 = WC / Total Assets
X2 = RE / Total Assets
X3 = EBIT / Total Assets
X4 = Market Cap / Total Liabilities
X5 = Sales / Total Assets

// Z > 2.99 → Safe zone
// Z < 1.81 → Distress zone

Вклад компонентов

Z-score Альтмана — модель прогнозирования банкротства, разработанная Эдвардом Альтманом в 1968 году. Комбинирует 5 финансовых показателей для оценки кредитоспособности компании.

Z = 1.2×X1 + 1.4×X2 + 3.3×X3 + 0.6×X4 + 1.0×X5

  • Z > 2.99 — безопасная зона (вероятность банкротства минимальна)
  • 1.81 ≤ Z ≤ 2.99 — серая зона (неопределённость)
  • Z < 1.81 — зона бедствия (высокая вероятность банкротства)
  • Модель разработана для публичных производственных компаний
