Задание 1.1. Сравнение инвестиционных предложений

Банк A: депозит 1 000 000 руб. под 10% годовых, капитализация ежегодно, 5 лет.
Банк B: депозит 1 000 000 руб. под 9,5% годовых, капитализация ежемесячно, 5 лет.

ЯчейкаСодержаниеФормула
B2Вклад-1 000 000
B3Ставка (годовая)10%
B4Период (лет)5
B5FV (Банк A)=БС(B3;B4;0;B2)
C3Ставка (месячная)=9,5%/12
C4Периодов=5*12
C5FV (Банк B)=БС(C3;C4;0;B2)

Ожидаемый результат:

  • Банк A:
    $$1\,000\,000 \times (1{,}10)^5 = 1\,610\,510 \text{ руб.}$$
  • Банк B:
    $$1\,000\,000 \times (1 + 0{,}095/12)^{60} = 1\,600\,527 \text{ руб.}$$

Банк А выгоднее, несмотря на менее частую капитализацию, из-за более высокой ставки.

Задание 2.2. Профиль NPV (график)

Построить зависимость NPV от ставки дисконтирования (0%-30%):

  1. В столбце A введите ставки: 0%, 2%, 4%, ..., 30%.
  2. В столбце B рассчитайте NPV Alpha: =ЧПС(A14;$B$3:$B$7)+$B$2.
  3. В столбце C рассчитайте NPV Beta аналогично.
  4. Выделите диапазон A:C и вставьте линейный график с двумя сериями.
  5. Точки пересечения с осью X -- это IRR каждого проекта.

График профиля NPV наглядно показывает, при какой ставке один проект становится выгоднее другого (точка пересечения кривых).

Рассчитать финансовые коэффициенты, построить Du Pont декомпозицию и Z-score Альтмана.

Тема 7.1

Задание 5.1. Панель коэффициентов (Dashboard)

Создать лист с автоматическим расчётом коэффициентов из исходных данных.

БлокПоказателиУсловное форматирование
ЛиквидностьCurrent, Quick, Cash RatioЗелёный (норма) / Красный
РентабельностьROA, ROE, ROS, EBITDA маржаГрадиент (больше = зеленее)
УстойчивостьАвтономии, D/E, ICRЗелёный / Жёлтый / Красный

Условное форматирование для Current Ratio:

  • Значение < 1,0 → красный фон
  • Значение 1,0-1,5 → жёлтый
  • Значение > 1,5 → зелёный

Задание 5.3. Z-score Альтмана

ЯчейкаФормулаОписание
X1=NWC/АктивыЛиквидность
X2=Нераспр_прибыль/АктивыНакопленная прибыльность
X3=EBIT/АктивыОперационная эффективность
X4=Рын_СК/ОбязательстваРыночная устойчивость
X5=Выручка/АктивыОборачиваемость
Z=1.2*X1+1.4*X2+3.3*X3+0.6*X4+1.0*X5Z-score
Зона=ЕСЛИ(Z>2.99;"Безопасная"; ЕСЛИ(Z>1.81;"Серая";"Опасная"))Интерпретация

Интерпретация Z-score:

$$Z = 1{,}2 X_1 + 1{,}4 X_2 + 3{,}3 X_3 + 0{,}6 X_4 + 1{,}0 X_5$$
  • Z > 2,99 -- безопасная зона (банкротство маловероятно)
  • 1,81 < Z < 2,99 -- серая зона (неопределённость)
  • Z < 1,81 -- опасная зона (высокий риск банкротства)

Именованные диапазоны

Используйте именованные диапазоны для читаемости формул:

  • WACC вместо $B$10
  • Tax_Rate вместо $C$6

Создание: выделить ячейку → поле имени (слева от строки формул) → ввести имя.

Защита модели

  • Защита листа: Рецензирование → Защитить лист (разблокировать только входные ячейки)
  • Проверка данных: Данные → Проверка данных (ограничить допустимые значения)
  • Цветовое кодирование: Входные данные -- синий шрифт, формулы -- чёрный
