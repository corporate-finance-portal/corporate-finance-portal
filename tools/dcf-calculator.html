Калькулятор DCF

Оцените стоимость компании методом дисконтированных денежных потоков с терминальной стоимостью, визуализацией структуры EV и пресетами российских эмитентов.

// Discounted Cash Flow Model
EV = ∑ FCFt / (1 + WACC)t + TV / (1 + WACC)n

// Терминальная стоимость (Gordon Growth):
TV = FCFn × (1 + g) / (WACC - g)

// где:
FCF  = свободный денежный поток
WACC = ставка дисконтирования
g    = долгосрочный темп роста
Обычно 2-5%, не выше номинального ВВП

Структура Enterprise Value

DCF (Discounted Cash Flow) — модель дисконтированных денежных потоков. Основной метод фундаментальной оценки стоимости компании. Стоимость определяется как сумма приведённых будущих денежных потоков.

EV = ∑t=1..n FCFt / (1 + WACC)t + TV / (1 + WACC)n

  • TV обычно составляет 60-80% от EV — это норма для DCF
  • Результат чувствителен к выбору g и WACC
  • Требование: WACC > g, иначе модель не имеет смысла
