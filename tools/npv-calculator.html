Калькулятор NPV

Рассчитайте чистую приведённую стоимость проекта с визуализацией денежных потоков, IRR, индексом прибыльности и дисконтированным сроком окупаемости.

// Чистая приведённая стоимость
NPV = ∑ CFt / (1 + r)t

// Дополнительные метрики:
PI  = PV(CF) / I0
IRR = r при NPV = 0
DPP = t при ∑PV(CF) ≥ 0

// Критерий: NPV > 0 → проект принять
Обычно равна WACC компании

Накопленная приведённая стоимость

NPV (Net Present Value) — чистая приведённая стоимость. Показывает прирост стоимости компании при реализации проекта. Если NPV > 0, проект создаёт стоимость и его следует принять.

NPV = -I0 + ∑t=1..n CFt / (1 + r)t

  • IRR — ставка дисконтирования, при которой NPV = 0
  • PI — отношение PV денежных потоков к начальным инвестициям (PI > 1 → проект выгоден)
  • DPP — число лет до полного возврата инвестиций с учётом дисконтирования
