Калькулятор облигаций

Цена, доходность, дюрация и денежные потоки. Рассчитайте справедливую стоимость облигации, YTM, дюрацию Маколея и модифицированную дюрацию с визуализацией.

P = ∑ C / (1 + r)t + FV / (1 + r)n

C    = купонный платёж
r    = YTM / частота
FV   = номинал (face value)
n    = кол-во купонных периодов

Оценка облигаций — определение справедливой стоимости как суммы дисконтированных будущих денежных потоков (купонов и номинала).

Цена облигации:

$P = \sum_{t=1}^{n} \frac{C}{(1+r)^t} + \frac{FV}{(1+r)^n}$

Дюрация Маколея:

$D_{Mac} = \frac{1}{P}\sum_{t=1}^{n} \frac{t \cdot CF_t}{(1+r)^t}$

Модифицированная дюрация:

$D_{mod} = \frac{D_{Mac}}{1 + r/m}$

  • C — купонный платёж за период = Номинал × Купон / Частота
  • r — доходность за период = YTM / Частота
  • n — общее число купонных периодов
  • m — частота купонных выплат в год
  • FV — номинал (возвращается при погашении)

Образовательный инструмент. Калькулятор предназначен для учебных целей:

  • Не учитывает накопленный купонный доход (НКД / accrued interest)
  • Предполагает, что следующий купон ровно через один купонный период
  • Для реальных расчётов используйте moex.com/bondization
