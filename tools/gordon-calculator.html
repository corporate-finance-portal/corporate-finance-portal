// Gordon Growth Model (DDM)
P0 = D1 / (Ke - g)

// где:
D1 = ожидаемый дивиденд
Ke = требуемая доходность
g   = темп роста дивидендов

DY = D1 / P0 = Ke - g

Модель Гордона (Gordon Growth Model, GGM) — модель дисконтирования дивидендов с постоянным темпом роста. Используется для оценки стоимости акций компаний со стабильной дивидендной политикой.

P0 = D1 / (Ke - g)

  • Требование: Ke > g, иначе модель не работает
  • Модель предполагает постоянный темп роста дивидендов навсегда
  • Лучше подходит для зрелых компаний с устойчивой дивидендной политикой
  • Для компаний роста лучше использовать DCF
