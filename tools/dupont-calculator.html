Анализ Дюпона

Разложите ROE на три компонента — маржу, оборачиваемость и леверидж, чтобы понять, за счёт чего компания генерирует рентабельность собственного капитала.

// DuPont Identity (3-factor)
ROE = Margin × Turnover × Leverage

// где:
Margin   = NI / Revenue
Turnover = Revenue / Assets
Leverage = Assets / Equity

// Проверка:
ROE = NI / Equity = Margin × Turnover × Leverage

Компоненты ROE

Вклад факторов (нормализованный)

Анализ Дюпона (DuPont Analysis) — метод декомпозиции ROE на три мультипликативных фактора, позволяющий понять источники рентабельности собственного капитала.

ROE = (NI / Rev) × (Rev / Assets) × (Assets / Equity)

  • Маржа — эффективность управления затратами (сколько прибыли с рубля выручки)
  • Оборачиваемость — эффективность использования активов (сколько выручки на рубль активов)
  • Леверидж — степень использования заёмного финансирования (чем выше, тем больше долга)
  • Высокий ROE за счёт левериджа — сигнал повышенного финансового риска
