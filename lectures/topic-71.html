1. Основные формы финансовой отчётности

ОтчётЧто показываетПериод
Бухгалтерский баланс (Balance Sheet)Активы, обязательства, капитал на датуНа момент
Отчёт о фин. результатах (P&L)Выручка, расходы, прибыль за периодЗа период
Отчёт о движении ДС (Cash Flow)Поступления и выплаты денегЗа период

Балансовое уравнение:

$$\text{Активы} = \text{Обязательства} + \text{Собственный капитал}$$

2. Методы анализа отчётности

2.1. Горизонтальный анализ (трендовый)

Сравнение показателей во времени:

  • Абсолютное изменение: $\Delta = X_t - X_{t-1}$
  • Темп роста: $\frac{X_t}{X_{t-1}} \times 100\%$
  • Темп прироста: $\frac{X_t - X_{t-1}}{X_{t-1}} \times 100\%$

2.2. Вертикальный анализ (структурный)

Доля каждой статьи в итоге:

  • В балансе: % от валюты баланса;
  • В P&L: % от выручки.

4. Модель Du Pont

Модель Du Pont — декомпозиция ROE для выявления драйверов доходности.

4.1. Трёхфакторная модель

$$ROE = \underbrace{\frac{NI}{\text{Выручка}}}_{ROS} \times \underbrace{\frac{\text{Выручка}}{\text{Активы}}}_{TAT} \times \underbrace{\frac{\text{Активы}}{\text{СК}}}_{EM}$$

$ROS$ — маржа (эффективность); $TAT$ — оборачиваемость (интенсивность); $EM$ — мультипликатор СК (леверидж).

4.2. Пятифакторная модель

$$ROE = \frac{NI}{EBT} \times \frac{EBT}{EBIT} \times \frac{EBIT}{\text{Выручка}} \times \frac{\text{Выручка}}{\text{Активы}} \times \frac{\text{Активы}}{\text{СК}}$$
ФакторЧто отражает
$NI/EBT$Налоговое бремя
$EBT/EBIT$Процентное бремя
$EBIT/\text{Выручка}$Операционная маржа
$\text{Выручка}/\text{Активы}$Оборачиваемость
$\text{Активы}/\text{СК}$Финансовый рычаг

5. Модель Альтмана (Z-score)

Прогнозирование вероятности банкротства на основе финансовых коэффициентов.

5.1. Модель для публичных компаний (1968)

$$Z = 1{,}2 X_1 + 1{,}4 X_2 + 3{,}3 X_3 + 0{,}6 X_4 + 1{,}0 X_5$$
ПеременнаяФормулаЧто отражает
$X_1$$\frac{NWC}{\text{Активы}}$Ликвидность
$X_2$$\frac{\text{Нераспр. прибыль}}{\text{Активы}}$Накопленная прибыль
$X_3$$\frac{EBIT}{\text{Активы}}$Рентабельность
$X_4$$\frac{\text{Рын. стоимость СК}}{\text{Обязательства}}$Платёжеспособность
$X_5$$\frac{\text{Выручка}}{\text{Активы}}$Деловая активность

Интерпретация Z-score:

Z-scoreЗонаВероятность банкротства
$Z > 2{,}99$БезопаснаяНизкая
$1{,}81 < Z < 2{,}99$СераяНеопределённая
$Z < 1{,}81$ОпаснаяВысокая

5.2. Модель для непубличных компаний (1983)

$$Z' = 0{,}717 X_1 + 0{,}847 X_2 + 3{,}107 X_3 + 0{,}420 X_4' + 0{,}998 X_5$$

$X_4' = \frac{\text{Балансовая стоимость СК}}{\text{Обязательства}}$ (вместо рыночной).

Границы: $Z' > 2{,}90$ — безопасно; $Z' < 1{,}23$ — опасно.

6. Кредитный анализ

Оценка способности компании обслуживать долг.

ПоказательФормулаНорма
ICR$\frac{EBIT}{\text{Проценты}}$> 3,0
DSCR$\frac{EBITDA}{\text{Проценты} + \text{Осн. долг}}$> 1,2
Долг/EBITDA$\frac{\text{Общий долг}}{EBITDA}$< 3,0
Чистый долг/EBITDA$\frac{\text{Долг} - \text{ДС}}{EBITDA}$< 2,5

Примеры расчётов

Пример 1. Комплексный коэффициентный анализ

Компания «Гамма» (млн руб.):

ПоказательЗначение
Выручка2 400
Себестоимость1 680
EBIT360
Чистая прибыль240
Проценты80
ОА / Запасы / ДС800 / 300 / 100
Активы2 000
КО / СК / Обязательства500 / 1 000 / 1 000

Ликвидность:

  • Current Ratio $= 800/500 = 1{,}60$ — норма
  • Quick Ratio $= (800-300)/500 = 1{,}00$ — норма
  • Cash Ratio $= 100/500 = 0{,}20$ — на нижней границе

Рентабельность:

  • ROA $= 240/2\,000 = 12{,}0\%$
  • ROE $= 240/1\,000 = 24{,}0\%$
  • ROS $= 240/2\,400 = 10{,}0\%$

Финансовая устойчивость:

  • Автономии $= 1\,000/2\,000 = 0{,}50$ — на границе
  • D/E $= 1\,000/1\,000 = 1{,}0$ — на границе
  • ICR $= 360/80 = 4{,}5$ — норма

Пример 2. Du Pont анализ

Используя данные «Гаммы»:

$$ROE = \underbrace{\frac{240}{2\,400}}_{10\%} \times \underbrace{\frac{2\,400}{2\,000}}_{1{,}2} \times \underbrace{\frac{2\,000}{1\,000}}_{2{,}0} = 0{,}10 \times 1{,}2 \times 2{,}0 = 24\%$$

Вывод: ROE = 24% достигается за счёт умеренной маржи (10%), хорошей оборачиваемости (1,2) и значительного рычага (2,0). Высокий рычаг означает зависимость от долга.

Пример 3. Z-score Альтмана

Компания «Дельта» (публичная, млн руб.): NWC = 200, Активы = 1 500, Нераспр. прибыль = 400, EBIT = 180, Рын. стоимость СК = 900, Обязательства = 600, Выручка = 2 100.

$$X_1 = 200/1\,500 = 0{,}133; \quad X_2 = 400/1\,500 = 0{,}267; \quad X_3 = 180/1\,500 = 0{,}120$$
$$X_4 = 900/600 = 1{,}500; \quad X_5 = 2\,100/1\,500 = 1{,}400$$
$$Z = 1{,}2 \times 0{,}133 + 1{,}4 \times 0{,}267 + 3{,}3 \times 0{,}120 + 0{,}6 \times 1{,}500 + 1{,}0 \times 1{,}400$$
$$= 0{,}160 + 0{,}374 + 0{,}396 + 0{,}900 + 1{,}400 = 3{,}23$$

$Z = 3{,}23 > 2{,}99$ → безопасная зона, вероятность банкротства низкая.

Практические задания

Задание 1. Коэффициентный анализ

Выручка = 1 800, Себестоимость = 1 260, EBIT = 270, ЧП = 162, Проценты = 54 (млн руб.). ОА = 600, Запасы = 200, КО = 400, Активы = 1 500, СК = 800.

Рассчитайте: Current Ratio, Quick Ratio, ROA, ROE, ROS, ICR, коэффициент автономии.

Решение:

  • Current Ratio $= 600/400 = 1{,}50$
  • Quick Ratio $= (600-200)/400 = 1{,}00$
  • ROA $= 162/1\,500 = 10{,}8\%$
  • ROE $= 162/800 = 20{,}3\%$
  • ROS $= 162/1\,800 = 9{,}0\%$
  • ICR $= 270/54 = 5{,}0$
  • Автономии $= 800/1\,500 = 0{,}53$
Задание 2. Du Pont

Используя данные Задания 1, разложите ROE по трёхфакторной модели.

Решение:

$$ROE = \frac{162}{1\,800} \times \frac{1\,800}{1\,500} \times \frac{1\,500}{800} = 0{,}09 \times 1{,}20 \times 1{,}875 = 20{,}3\%$$

Драйверы: маржа 9%, оборачиваемость 1,2, рычаг 1,875.

Задание 3. Z-score (непубличная компания)

NWC = 150, Активы = 1 000, Нераспр. прибыль = 250, EBIT = 100, Балансовая СК = 500, Обязательства = 500, Выручка = 1 400 (млн руб.).

Оцените вероятность банкротства по модели $Z'$.

Решение:

$$X_1 = 0{,}150; \quad X_2 = 0{,}250; \quad X_3 = 0{,}100; \quad X_4' = 1{,}000; \quad X_5 = 1{,}400$$
$$Z' = 0{,}717 \times 0{,}15 + 0{,}847 \times 0{,}25 + 3{,}107 \times 0{,}10 + 0{,}420 \times 1{,}0 + 0{,}998 \times 1{,}4$$
$$= 0{,}108 + 0{,}212 + 0{,}311 + 0{,}420 + 1{,}397 = 2{,}45$$

$1{,}23 < Z' = 2{,}45 < 2{,}90$ → серая зона, требуется дополнительный анализ.
