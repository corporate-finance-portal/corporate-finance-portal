1. Зачем оценивать стоимость компании

Оценка стоимости компании (Business Valuation) — определение справедливой рыночной стоимости бизнеса или его доли на основе финансовых данных и прогнозов.

Ситуации, требующие оценки:

  • M&A — покупка/продажа бизнеса;
  • IPO — определение цены размещения акций;
  • Привлечение инвестиций — обоснование стоимости для инвесторов;
  • Судебные споры — раздел имущества, банкротство;
  • Управление стоимостью (VBM) — стратегические решения менеджмента.

2. Три подхода к оценке

ПодходСутьКогда применять
ДоходныйСтоимость = PV будущих доходовДействующий бизнес с прогнозируемыми потоками
СравнительныйСтоимость по аналогии с похожими компаниямиЕсть публичные аналоги или сделки
ЗатратныйСтоимость = рыночная стоимость активов - обязательстваЛиквидация, имущественные компании

В корпоративных финансах доходный подход (DCF) считается основным, сравнительный — проверочным.

3. Метод DCF (Discounted Cash Flow)

DCF — ключевой метод оценки. Стоимость компании равна приведённой стоимости всех будущих свободных денежных потоков.

$$V_0 = \sum_{t=1}^{n} \frac{FCF_t}{(1 + WACC)^t} + \frac{TV_n}{(1 + WACC)^n}$$

$V_0$ — Enterprise Value; $FCF_t$ — свободный поток в году $t$; $WACC$ — ставка дисконтирования; $TV_n$ — терминальная стоимость; $n$ — прогнозный период (5-10 лет).

3.1. Свободный денежный поток (FCF)

FCFF (Free Cash Flow to Firm) — поток для всех поставщиков капитала:

$$FCFF = EBIT \times (1 - T) + DA - CAPEX - \Delta NWC$$

$EBIT$ — прибыль до процентов и налогов; $T$ — ставка налога; $DA$ — амортизация; $CAPEX$ — капитальные затраты; $\Delta NWC$ — изменение чистого оборотного капитала.

FCFE (Free Cash Flow to Equity) — поток только для акционеров:

$$FCFE = FCFF - \text{Проценты} \times (1 - T) + \Delta \text{Долг}$$
ПотокСтавка дисконтированияРезультат
FCFFWACCEnterprise Value (EV)
FCFE$k_e$ (стоимость собственного капитала)Equity Value

3.2. Терминальная стоимость (Terminal Value)

Терминальная стоимость отражает стоимость бизнеса после прогнозного периода. Обычно составляет 60-80% от общей стоимости компании.

Метод 1 — модель Гордона (Gordon Growth Model):

$$TV_n = \frac{FCF_{n+1}}{WACC - g} = \frac{FCF_n \times (1 + g)}{WACC - g}$$

$g$ — долгосрочный темп роста FCF (обычно 2-3%, не выше темпа роста ВВП).

Метод 2 — мультипликатор выхода (Exit Multiple):

$$TV_n = EBITDA_n \times \text{EV/EBITDA}_{\text{отрасль}}$$

4. Сравнительная оценка (мультипликаторы)

Метод сравнительной оценки определяет стоимость через соотношения цены и финансовых показателей компаний-аналогов.

4.1. Основные мультипликаторы

МультипликаторФормулаПрименение
P/E$\frac{\text{Цена акции}}{EPS}$Прибыльные компании
EV/EBITDA$\frac{EV}{EBITDA}$Сравнение с разным долгом
P/B$\frac{\text{Цена}}{BV \text{ на акцию}}$Банки, капиталоёмкие отрасли
P/S$\frac{\text{Цена}}{\text{Выручка на акцию}}$Убыточные быстрорастущие компании
EV/Sales$\frac{EV}{\text{Выручка}}$Ранние стадии, стартапы

4.2. Enterprise Value (EV)

$$EV = \text{Рыночная капитализация} + \text{Чистый долг}$$
$$\text{Чистый долг} = \text{Долг} - \text{Денежные средства}$$

4.3. Алгоритм сравнительной оценки

  1. Выбрать компании-аналоги (отрасль, размер, география, темп роста);
  2. Рассчитать мультипликаторы для каждого аналога;
  3. Определить медианный мультипликатор (медиана устойчивее среднего);
  4. Применить к показателям оцениваемой компании;
  5. Получить диапазон оценки (не точку).

5. EVA (Economic Value Added)

EVA — экономическая добавленная стоимость, показывает создаёт ли компания стоимость сверх стоимости капитала.

$$EVA = NOPAT - WACC \times IC$$

$NOPAT = EBIT \times (1 - T)$ — чистая операционная прибыль после налогов; $IC$ — инвестированный капитал (СК + долг); $WACC$ — средневзвешенная стоимость капитала.

Интерпретация:

  • $EVA > 0$ — компания создаёт стоимость (доходность > стоимости капитала);
  • $EVA < 0$ — компания разрушает стоимость;
  • $EVA = 0$ — компания зарабатывает ровно столько, сколько требуют инвесторы.

MVA (Market Value Added):

$$MVA = \text{Рыночная стоимость} - IC = \sum_{t=1}^{\infty} \frac{EVA_t}{(1+WACC)^t}$$

6. От Enterprise Value к цене акции

$$\text{Equity Value} = EV - \text{Чистый долг} - \text{Привилег. акции} - \text{Доля меньшинства}$$
$$\text{Цена акции} = \frac{\text{Equity Value}}{\text{Количество акций в обращении}}$$

Терминальная стоимость (модель Гордона, $g = 2\%$):

$$TV_5 = \frac{524 \times 1{,}02}{0{,}12 - 0{,}02} = \frac{534{,}48}{0{,}10} = 5\,344{,}8 \text{ млн руб.}$$

Приведённая стоимость:

$$PV(FCF) = \frac{330}{1{,}12} + \frac{378}{1{,}12^2} + \frac{426}{1{,}12^3} + \frac{479}{1{,}12^4} + \frac{524}{1{,}12^5} = 1\,500{,}9 \text{ млн}$$
$$PV(TV) = \frac{5\,344{,}8}{1{,}12^5} = 3\,032{,}8 \text{ млн руб.}$$
$$EV = 1\,500{,}9 + 3\,032{,}8 = 4\,533{,}7 \text{ млн руб.}$$

Чистый долг = 800 млн руб., акций 10 млн шт.:

$$\text{Equity Value} = 4\,533{,}7 - 800 = 3\,733{,}7 \text{ млн руб.}$$
$$\text{Цена акции} = \frac{3\,733{,}7}{10} = 373{,}4 \text{ руб.}$$

Обратите внимание: терминальная стоимость составляет $3\,032{,}8 / 4\,533{,}7 = 66{,}9\%$ от EV — типичное значение для DCF-модели.

Пример 3. EVA

Компания: $NOPAT = 450$ млн руб., $IC = 3\,000$ млн руб., $WACC = 12\%$.

$$EVA = 450 - 0{,}12 \times 3\,000 = 450 - 360 = +90 \text{ млн руб.}$$

Вывод: компания создаёт стоимость. Доходность на капитал $ROIC = 450/3\,000 = 15\% > WACC = 12\%$.

Практические задания

Задание 1. DCF-оценка

FCFF: год 1 = 200, год 2 = 230, год 3 = 260 млн руб. WACC = 14%, $g = 2\%$. Чистый долг = 300 млн руб.

Найти: EV, Equity Value.

Решение:

$$TV_3 = \frac{260 \times 1{,}02}{0{,}14 - 0{,}02} = \frac{265{,}2}{0{,}12} = 2\,210 \text{ млн руб.}$$
$$PV(FCF) = \frac{200}{1{,}14} + \frac{230}{1{,}14^2} + \frac{260}{1{,}14^3} = 175{,}4 + 177{,}0 + 175{,}5 = 527{,}9$$
$$PV(TV) = \frac{2\,210}{1{,}14^3} = 1\,491{,}7$$
$$EV = 527{,}9 + 1\,491{,}7 = 2\,019{,}6 \text{ млн руб.}$$
$$\text{Equity Value} = 2\,019{,}6 - 300 = \mathbf{1\,719{,}6} \text{ млн руб.}$$
Задание 2. Мультипликаторы

EBITDA = 500 млн руб., EPS = 45 руб. Медианы: EV/EBITDA = 7,5; P/E = 13. Чистый долг = 1 200 млн руб., акций = 20 млн.

Найти: диапазон цены акции.

Решение:

По EV/EBITDA:

$$EV = 500 \times 7{,}5 = 3\,750; \quad Equity = 3\,750 - 1\,200 = 2\,550; \quad P = 2\,550/20 = 127{,}5 \text{ руб.}$$

По P/E:

$$P = 45 \times 13 = 585 \text{ руб.}$$

Диапазон: 127,5-585 руб. Большой разброс указывает на необходимость проверить причины расхождения.

Задание 3. EVA

$EBIT = 800$ млн руб., $T = 20\%$, $IC = 5\,000$ млн руб., $WACC = 11\%$.

Решение:

$$NOPAT = 800 \times (1 - 0{,}20) = 640 \text{ млн руб.}$$
$$EVA = 640 - 0{,}11 \times 5\,000 = 640 - 550 = +90 \text{ млн руб.}$$
$$ROIC = 640 / 5\,000 = 12{,}8\% > 11\% = WACC$$

Компания создаёт стоимость.
