Оценка инвестиционных
проектов: NPV, IRR, PI, DPP

1. Инвестиционный процесс корпорации

Инвестиционные решения (Capital Budgeting) — решения о вложении капитала в долгосрочные активы с целью получения будущих денежных потоков. Это одни из самых важных решений в корпоративных финансах.

Инвестиционные решения критически важны, поскольку они:

  • определяют будущую стоимость компании;
  • связывают значительные ресурсы на длительный срок;
  • в большинстве случаев необратимы.

Инвестиции классифицируются по нескольким критериям:

  • По характеру: реальные (в основные средства, НИОКР) и финансовые (в ценные бумаги);
  • По взаимоотношению: независимые (принятие одного не влияет на другой) и взаимоисключающие (можно выбрать только один);
  • По обязательности: обязательные (экология, безопасность) и дискреционные (расширение, модернизация).

2. Денежные потоки инвестиционного проекта

При оценке проекта используются приростные (инкрементальные) денежные потоки — разница между потоками компании с проектом и без него.

Основные компоненты:

  • Начальные инвестиции (IC) — первоначальные вложения (отрицательный поток в момент $t = 0$);
  • Операционные потоки (CF) — чистые денежные поступления в каждом периоде;
  • Терминальный поток — ликвидационная стоимость активов по окончании проекта.

Правила формирования денежных потоков:

  1. Учитывать только приростные потоки (не суммарные);
  2. Учитывать альтернативные издержки (opportunity cost);
  3. Не учитывать невозвратные затраты (sunk costs);
  4. Учитывать побочные эффекты (каннибализация, синергия);
  5. Учитывать изменения в оборотном капитале.

3. Методы оценки инвестиционных проектов

3.1. Чистая приведённая стоимость — NPV (Net Present Value)

NPV — сумма дисконтированных денежных потоков проекта за вычетом первоначальных инвестиций:

$$NPV = \sum_{t=1}^{n} \frac{CF_t}{(1 + r)^t} - IC$$

$CF_t$ — чистый денежный поток в году $t$, $r$ — ставка дисконтирования (обычно WACC), $IC$ — начальные инвестиции, $n$ — срок проекта.

Правило принятия решения:

  • $NPV > 0$ — проект создаёт стоимость — принять
  • $NPV < 0$ — проект разрушает стоимость — отклонить
  • $NPV = 0$ — проект безразличен

Достоинства NPV:

  • Учитывает временну́ю стоимость денег;
  • Показывает абсолютный прирост стоимости компании;
  • Обладает свойством аддитивности: $NPV_{\text{портфеля}} = \sum NPV_i$;
  • Является главным критерием при конфликте с другими показателями.

3.2. Внутренняя норма доходности — IRR (Internal Rate of Return)

IRR — ставка дисконтирования, при которой NPV проекта равен нулю. Это «доходность самого проекта».

$$\sum_{t=1}^{n} \frac{CF_t}{(1 + IRR)^t} = IC$$

IRR показывает максимальную стоимость капитала, при которой проект остаётся безубыточным.

Правило: $IRR > WACC$ — принять; $IRR < WACC$ — отклонить.

Метод линейной интерполяции:

$$IRR \approx r_1 + \frac{NPV_1}{NPV_1 - NPV_2} \times (r_2 - r_1)$$

$r_1$ — ставка, при которой $NPV_1 > 0$; $r_2$ — ставка, при которой $NPV_2 < 0$.

Ограничения IRR:

  • Может давать несколько значений при нестандартных потоках (смена знаков);
  • При взаимоисключающих проектах может конфликтовать с NPV (приоритет у NPV);
  • Не учитывает масштаб проекта.

3.3. Индекс рентабельности — PI (Profitability Index)

$$PI = \frac{\sum_{t=1}^{n} \frac{CF_t}{(1 + r)^t}}{IC} = \frac{PV_{\text{притоков}}}{IC}$$

Правило: $PI > 1$ — принять; $PI < 1$ — отклонить.

PI показывает, сколько рублей приведённой стоимости генерирует каждый рубль инвестиций. Это относительный показатель, особенно полезный при ранжировании проектов в условиях ограниченного бюджета (рационирование капитала).

3.4. Дисконтированный срок окупаемости — DPP

DPP — период, за который кумулятивный дисконтированный денежный поток сравняется с начальными инвестициями.

$$DPP = t^* + \frac{IC - \sum_{t=1}^{t^*} \frac{CF_t}{(1+r)^t}}{\frac{CF_{t^*+1}}{(1+r)^{t^*+1}}}$$

$t^*$ — последний год, в котором кумулятивный PV ещё меньше IC.

Достоинства: учитывает временну́ю стоимость денег (в отличие от простого PP). Недостатки: не учитывает потоки после окупаемости, не показывает величину создаваемой стоимости.

3.5. Модифицированная IRR — MIRR

MIRR устраняет проблему множественных IRR, предполагая реинвестирование промежуточных потоков по ставке WACC:

$$MIRR = \left(\frac{\sum_{t=1}^{n} CF_t^{+} \times (1+r)^{n-t}}{\sum_{t=0}^{n} \frac{|CF_t^{-}|}{(1+r)^t}}\right)^{1/n} - 1$$

4. Конфликт критериев NPV и IRR

При оценке взаимоисключающих проектов NPV и IRR могут давать противоположные ранжирования из-за:

  • различий в масштабе проектов;
  • различий во временно́м распределении потоков;
  • различий в предположении о ставке реинвестирования.

Правило: при конфликте приоритет всегда у NPV, поскольку NPV непосредственно измеряет прирост стоимости компании.

5. Анализ чувствительности

Анализ чувствительности показывает, как изменяется NPV при изменении одного из ключевых параметров (ставка дисконтирования, объём продаж, себестоимость). Это позволяет оценить устойчивость проекта и выявить критические переменные.

Пример 3. Расчёт IRR методом интерполяции

Условие: IC = 4 000 000 руб. Потоки: CF₁ = 1 200 000; CF₂ = 1 500 000; CF₃ = 1 800 000; CF₄ = 1 600 000.

При $r_1 = 15\%$: $NPV_1 = +276\,028$ руб. При $r_2 = 20\%$: $NPV_2 = -145\,062$ руб.

$$IRR \approx 15\% + \frac{276\,028}{276\,028 + 145\,062} \times (20\% - 15\%)$$
$$IRR \approx 15\% + \frac{276\,028}{421\,090} \times 5\% \approx 15\% + 0{,}656 \times 5\% \approx 18{,}28\%$$

Если WACC компании = 14%, проект принимается ($IRR = 18{,}28\% > 14\%$).

Задание 3. Расчёт IRR и принятие решения

Решение:

При $r_1 = 15\%$: $NPV_1 = +276\,028$. При $r_2 = 20\%$: $NPV_2 = -145\,062$.

$$IRR \approx 15\% + \frac{276\,028}{421\,090} \times 5\% \approx 18{,}28\%$$

$IRR = 18{,}28\% > WACC = 14\%$ — проект принимается.
