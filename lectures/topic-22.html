1. Гипотеза эффективного рынка (EMH)

Гипотеза эффективного рынка (Efficient Market Hypothesis) — цены активов полностью отражают доступную информацию.

ФормаИнформация в ценахСледствие
СлабаяИсторическая (прошлые цены, объёмы)Технический анализ бесполезен
СредняяВся публичная информацияФундаментальный анализ бесполезен
СильнаяВся информация, включая инсайдерскуюНевозможно систематически обыгрывать рынок

На практике рынки в целом эффективны в средней форме, но отклонения (аномалии) существуют. CAPM предполагает эффективность рынка.

2. Линия рынка капиталов (CML)

CML (Capital Market Line) связывает доходность и общий риск ($\sigma$) эффективных портфелей:

$$E(r_p) = r_f + \frac{E(r_m) - r_f}{\sigma_m} \times \sigma_p$$

Наклон CML — «рыночная цена риска» (Sharpe Ratio рынка). Любой эффективный портфель — комбинация безрискового актива и рыночного портфеля.

3. Модель CAPM

CAPM (Capital Asset Pricing Model) — модель оценки капитальных активов:

$$E(r_i) = r_f + \beta_i \times (E(r_m) - r_f)$$

$r_f$ — безрисковая ставка; $\beta_i$ — бета-коэффициент актива; $(E(r_m) - r_f)$ — рыночная премия за риск (ERP).

Предпосылки CAPM:

  1. Инвесторы рациональны и не склонны к риску;
  2. Рынок совершенен (нет налогов, транзакционных издержек);
  3. Все инвесторы имеют одинаковые ожидания (однородные ожидания);
  4. Можно занимать и давать взаймы по безрисковой ставке;
  5. Все активы делимы и ликвидны.

4. Бета-коэффициент ($\beta$)

Бета — мера систематического риска актива:

$$\beta_i = \frac{Cov(r_i, r_m)}{Var(r_m)} = \frac{\rho_{i,m} \times \sigma_i}{\sigma_m}$$
$\beta$Тип активаХарактеристика
$\beta > 1$АгрессивныйРастёт/падает быстрее рынка
$\beta = 1$РыночныйДвижется синхронно с рынком
$0 < \beta < 1$ЗащитныйМенее волатилен, чем рынок
$\beta = 0$БезрисковыйНе зависит от рынка

Справочные бета для российских компаний (capm-navigator.ru):

  • Лукойл: $\beta \approx 0{,}95$ (почти рыночный)
  • Газпром: $\beta \approx 0{,}51$ (защитный)
  • Сбербанк: $\beta \approx 1{,}20$ (агрессивный)
  • Яндекс: $\beta \approx 1{,}35$ (агрессивный, технологический сектор)

5. Линия рынка ценных бумаг (SML)

SML (Security Market Line) — графическое представление CAPM: зависимость $E(r)$ от $\beta$.

  • Актив выше SML — недооценён (доходность выше справедливой → покупать);
  • Актив ниже SML — переоценён (доходность ниже справедливой → продавать).

Альфа Дженсена (Jensen's Alpha):

$$\alpha_i = r_i^{\text{факт}} - [r_f + \beta_i \times (r_m - r_f)]$$
  • $\alpha > 0$: управляющий обеспечил доходность выше ожидаемой по CAPM;
  • $\alpha < 0$: доходность ниже ожидаемой.

7. Ограничения и критика CAPM

  1. Предпосылки нереалистичны (совершенный рынок, однородные ожидания);
  2. $\beta$ нестабильна во времени (зависит от периода расчёта);
  3. Одного фактора ($\beta$) недостаточно для объяснения доходности;
  4. Проблема выбора рыночного портфеля (критика Ролла);
  5. На развивающихся рынках CAPM работает хуже.

8. Многофакторные модели

Модель Фама–Френча (3 фактора):

$$E(r_i) = r_f + \beta_1 \times (r_m - r_f) + \beta_2 \times SMB + \beta_3 \times HML$$
  • SMB (Small Minus Big) — премия за размер (малые компании доходнее крупных);
  • HML (High Minus Low) — премия за стоимость (компании с высоким B/M доходнее «компаний роста»).

Расширение: 5-факторная модель Фама–Френча добавляет рентабельность (RMW) и инвестиции (CMA).

Примеры расчётов

Пример 1. Стоимость собственного капитала для ПАО «Лукойл»

$r_f = 14{,}6\%$, $\beta = 0{,}95$, $ERP = 6\%$.

$$k_e = 14{,}6\% + 0{,}95 \times 6\% = 14{,}6\% + 5{,}7\% = 20{,}3\%$$

Акционеры Лукойла требуют не менее 20,3% годовой доходности.

Пример 2. Стоимость собственного капитала для ПАО «Газпром»

$r_f = 14{,}6\%$, $\beta = 0{,}51$, $ERP = 6\%$.

$$k_e = 14{,}6\% + 0{,}51 \times 6\% = 14{,}6\% + 3{,}06\% = 17{,}66\%$$

Газпром — защитный актив ($\beta < 1$), поэтому требуемая доходность ниже, чем у Лукойла.

Пример 3. Анализ по SML — актив переоценён или недооценён?

Актив: $\beta = 1{,}3$, фактическая доходность = 22%. Рынок: $r_f = 14{,}6\%$, $ERP = 6\%$.

Справедливая доходность по CAPM:

$$E(r) = 14{,}6\% + 1{,}3 \times 6\% = 14{,}6\% + 7{,}8\% = 22{,}4\%$$
$$\alpha = 22\% - 22{,}4\% = -0{,}4\%$$

Актив лежит чуть ниже SML → слегка переоценён (фактическая доходность ниже справедливой).

Практические задания

Задание 1. Расчёт требуемой доходности

$r_f = 15\%$, $\beta = 1{,}2$, рыночная премия = 5,5%.

Рассчитайте $k_e$ по CAPM.

Решение:

$$k_e = 15\% + 1{,}2 \times 5{,}5\% = 15\% + 6{,}6\% = 21{,}6\%$$
Задание 2. Определение $\beta$ из данных

Фактическая доходность акции = 19%, $r_f = 14\%$, $r_m = 20\%$.

Определите $\beta$, предполагая $\alpha = 0$.

Решение:

$$19\% = 14\% + \beta \times (20\% - 14\%)$$
$$5\% = \beta \times 6\%$$
$$\beta = 5/6 = 0{,}83$$
Задание 3. Оценка акций по SML

Акция X: $\beta = 0{,}8$, фактическая доходность = 20%. Акция Y: $\beta = 1{,}4$, фактическая доходность = 24%. $r_f = 14\%$, $ERP = 6\%$.

Какая акция недооценена?

Решение:

$$E(r_X) = 14\% + 0{,}8 \times 6\% = 18{,}8\%; \quad \alpha_X = 20\% - 18{,}8\% = +1{,}2\%$$
$$E(r_Y) = 14\% + 1{,}4 \times 6\% = 22{,}4\%; \quad \alpha_Y = 24\% - 22{,}4\% = +1{,}6\%$$

Обе акции выше SML ($\alpha > 0$) → недооценены. Y имеет большую альфу → более привлекательна.
