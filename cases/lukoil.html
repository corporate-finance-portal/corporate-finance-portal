Описание ситуации

ПАО «Лукойл» — одна из крупнейших частных нефтегазовых компаний мира. Компания рассматривает инвестиционный проект по модернизации нефтеперерабатывающего завода стоимостью 45 млрд руб. Для принятия решения необходимо рассчитать WACC компании и оценить проект методом NPV.

Цель кейса: рассчитать стоимость собственного капитала ($k_e$) методом CAPM, определить WACC, оценить проект по NPV, PI и DPP, провести анализ чувствительности.

Пошаговое решение

Шаг 1. Стоимость собственного капитала (CAPM)

Применяем модель CAPM для расчёта $k_e$:

$$k_e = r_f + \beta \times ERP = 14{,}6\% + 0{,}95 \times 6{,}0\% = 14{,}6\% + 5{,}7\% = 20{,}3\%$$

Шаг 2. Стоимость заёмного капитала (после налогов)

Учитываем налоговый щит:

$$k_d^{после} = k_d \times (1 - T) = 14{,}0\% \times (1 - 0{,}25) = 14{,}0\% \times 0{,}75 = 10{,}5\%$$

Шаг 3. Доли в структуре капитала

$$w_e = \frac{E}{V} = \frac{4\,500}{5\,000} = 0{,}90 \quad (90\%)$$
$$w_d = \frac{D}{V} = \frac{500}{5\,000} = 0{,}10 \quad (10\%)$$

Шаг 4. Расчёт WACC

$$WACC = w_d \times k_d^{после} + w_e \times k_e = 0{,}10 \times 10{,}5\% + 0{,}90 \times 20{,}3\%$$
$$WACC = 1{,}05\% + 18{,}27\% = \boxed{19{,}32\%}$$

Шаг 5. Дисконтирование денежных потоков

Дисконтируем каждый денежный поток по ставке WACC = 19,32%:

Шаг 6. Расчёт NPV

$$NPV = \sum_{t=1}^{7} \frac{CF_t}{(1 + WACC)^t} - IC = 31\,608 - 45\,000 = \boxed{-13\,392 \text{ млн руб.}}$$

Шаг 7. PI и DPP

Индекс прибыльности:

$$PI = \frac{\sum PV(CF_t)}{IC} = \frac{31\,608}{45\,000} = \boxed{0{,}702}$$

Дисконтированный срок окупаемости (DPP): кумулятивный PV не достигает 45 000 за 7 лет — проект не окупается в дисконтированных потоках.

Анализ чувствительности

Сценарий: рост $\beta$ до 1,2 (геополитические риски)

$$k_e = 14{,}6\% + 1{,}2 \times 6{,}0\% = 14{,}6\% + 7{,}2\% = 21{,}8\%$$
$$WACC = 0{,}10 \times 10{,}5\% + 0{,}90 \times 21{,}8\% = 1{,}05\% + 19{,}62\% = 20{,}67\%$$

При повышенном WACC (20,67%) NPV ухудшается до ~−15 800 млн руб., усиливая вывод об отклонении проекта.

При каком WACC проект станет рентабельным?

Проект выходит на NPV = 0 при WACC ≈ 14%. Это возможно при значительном снижении ключевой ставки ЦБ (что уменьшит $r_f$) или при увеличении доли заёмного финансирования с сохранением низкой $k_d$.

Выводы

  • NPV = −13 392 млн руб. < 0, PI = 0,702 < 1 — проект следует отклонить при текущей стоимости капитала.
  • Высокий WACC (19,32%) обусловлен: 90% собственного капитала (дорогой, не даёт налогового щита) и высокой безрисковой ставкой ($r_f$ = 14,6%).
  • Парадокс: компания с минимальным долгом имеет высокий WACC. Увеличение доли долга могло бы снизить WACC за счёт налогового щита, но увеличит финансовый риск.
  • Типичная ситуация для российских компаний в период высокой ключевой ставки ЦБ: при снижении ключевой ставки проект может стать рентабельным (NPV ≈ 0 при WACC ≈ 14%).
